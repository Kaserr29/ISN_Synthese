Adresse absolue VS adresse relative

L'Adresse en informatique

En informatique, lorsque le parle d'adresse d'un fichier, on désigne l'emplacement dans la mémoire où est stocké le fichier. Il s'agit en quelque sorte du chemin a emprunter pour pouvoir y accèder. On peut avoir besoin de cette adresse dans de nombreuses situations, particulièrement en programmation , par exemple dans un fichier HTML pour y inclure une feuille de style.

Il existe 2 type d'adresse.

Adresse relative

C'est une méthode qui est particulièrement utilisé en interne, par exemple en HTML. Elle consiste à se déplacer dans une arboressence de fichier par une suite d'instructions, un itinéraire, à partir d'un point de départ défini, tout comme un GPS pourrait vous guider sur les routes.

Exemple :

Considérons l'arborescence suivante : 

racine
	|-- index.html
	|-- javascript
	|-- images
	|	|-- chat
	|	|	|-- chat_trop_mignon.jpg
	|	|-- chien_rigolo.jpg
	|-- style
	|	|-- style.css	
	|-- pages
	|	|-- les_chiens.html
	|	|-- les_chats.html

Si je veux intégrer l'image chien_rigolo.jpg dans ma page les_chiens.html, je dois utiliser l'adresse relative suivante

	../images/chien_rigolo.jpg

Cette adresse se décompose de la façon suivante :

  • On est dans le dossier pages avec le fichier les_chiens.html
  • On recule d'un élément dans l'arborescence avec le "../", on est donc dans la racine
  • On entre dans le dossier images avec "/images"
  • On sélectionne l'image chien_rigolo.html avec "chien_rigolo.html"

Le chemin relatif est donc un moyen efficace de se déplacer dans une arborescence, à condition de connaitre le point de départ.

Adresse absolue

Le chemin absolu, contrairement à une adresse relative n'a pas besoin de point départ défini puisque que l'on partira toujours de la racine de l'arborescence, un peu comme votre adresse réelle. Il faut donc indiquer tout le chemin à parcourir depuis la racine jusqu'au fichier.

Cette méthode s'utilise en externe : l'adresse d'un site internet, comme "https://fr.wikipedia.org/wiki/World_Wide_Web" est une adresse absolue.

Exemple :

Considérons la même arborescence :

racine
	|-- index.html
	|-- javascript
	|-- images
	|	|-- chat
	|	|	|-- chat_trop_mignon.jpg
	|	|-- chien_rigolo.jpg
	|-- style
	|	|-- style.css	
	|-- pages
	|	|-- les_chiens.html
	|	|-- les_chats.html

Si l'on veut accéder à l'image chat_trop_mignon.jpg on utilisera l'adresse absolue suivante :

	racine/images/chat_trop_migion.jpg

Ici, contrairement à une adresse relative, on ne fait que avancer dans l'arborescence, on ne peut pas reculer.

Le chemin absolu est un moyen efficace pour accéder à une arborescence sans connaitre son point de départ.
