Processing

Introduction

Processing, anciennement appelé Pro55ing, est à la fois une bibliothèque Java et un environnement de développement, c’est à dire qu’il s’agit d’une version “simplifiée” de Java accompagnée d’un éditeur de texte et d’autres outils facilitant la programmation.

Ses créateurs : Benjamin Fry et Casey Reas, deux artistes américains, l’on conçu comme un outil d’apprentissage, libre et gratuit. Le logiciel est en effet simple d’utilisation et très diversifié dans ses applications : dessin 2D, 3D, son, vidéo … permettant aux débutants comme aux initiés de laisser libre cours à leurs créativités.

Structure de base d'un programme

En général, la structure de base d’un programme ce compose de 3 parties. Dans un premier temps, on déclare les variables dont on aura besoin dans le programme : elles seront des des variables dites globales. On utilise ensuite la fonction Setup(), dont tout le code contenu à l’intérieur de celle ci ne s'exécutera qu’une seule fois, au lancement du programme. Puis on utilise la fonction Draw(), qui exécutée 30 fois par seconde (valeur modifiable via la fonction framerate(int)).

Il faut également savoir que chaque instruction est séparée d’un point virgule “;” afin d’indiquer la fin d’une instruction à l’ordinateur et qu’il faut entourer les les blocs de code d’accolades “{ }”.

On peut également clarifier son code en y rajoutant des commentaires : la ligne précédée de // ou tout les texte contenue entre /* et */ sera ignoré lors de l’execution du code.

Exemple :

// Je suis ignoré !
/*
Moi aussi
*/
		
int variableGlobale = 0;
			
void setup() {
	println("Je ne m’exécute qu’une fois");
}
		
void draw() {
	println("Je m’exécute 30 fois par seconde");
	println(variableGlobale);
	variableGlobale++;
}

Les Variables et tableaux

Les variables permettent, en programmation de stocker des valeurs et effectuer des opérations sur celles-ci. Elles sont définies par :

  • Un type, permet de préciser le type de valeur contenue dans la variables :
    • int : nombre entier de -2 147 483 647 à 2 147 483
    • float : nombre décimal de -3,40282347*10^38 à -3,40282347*10^38 avec un précision limité
    • double : nombre décimal de -3,40282347*10^38 à 3,40282347*10^38 avec un plus grande précision que le float
    • boolean : variable pouvant avoir seulement 2 états : true (vrai) ou false (faux)
    • char : un seul caractère
    • String chaine de caractères
  • Un nom, défini par le développeur (à condition que le nom ne soit pas déjà utilisé pour une autre variable) il permet d'appeler la valeur de la variable
  • Une valeur, sur laquelle on pourra effectuer plusieurs opérations (surtout arithmétique pour les variables nombre)

Exemple :

int nbrEntier = 10;
int autreNbrEntier;
float nbrDecimal = 1.2;
char lettreA = ‘a’, lettreB = ‘b’;

void setup() {
	autreNbrEntier = nbrEntier*2;
	nbrEntier = autreNbrEntier/2 + 3;
	nbrDecimal = autreNbrEntier-nbrDecimal;
	lettreA = lettreB;
}

Il faut faire attention puisqu’il existe des variables prédéfinies en Processing (il ne faut donc pas réutiliser ces noms) comme par exemple width et height qui renvoie les dimensions de la fenêtre ou encore mouseX et mouseY qui renvoient les coordonnés de la souris.

Il existe également un type de variable très utile : le tableau. Elle permet de stocker plusieurs variables d’un même type sous forme de tableau à une ou plusieurs dimension, on accède donc aux valeurs via son ou ses indices dont le premier est 0. Il existe également des opérations que l’on peut directement effectuer sur ces tableaux.

Exemple :

int [] tableauNbrEntier = {0,4,5,1,2}; // tableau d’entier à 1 dimension
char [][] tableauDeCaractere = {{a,b},{a,g,j}}; // tableau de caractères à 2 dimensions
char lettre;
int nombre;
					
void setup() {
	nombre = tableauNbrEntier[2];
	tableauNbrEntier[1] = 5;
	lettre = tableauDeCaractere[0][1];
	tableauDeCaractere[1][3] = ‘z’;
	tableauDeCaractere = reverse(tableauDeCaractere);
}

Les Conditions et Boucles

La condition est un élément essentiel en programmation puisqu'il permet de n'effectuer une partie du code que si certaines conditions sont remplies. Pour ce faire, nous utilisons le mot clée if suivi de ladite condition entre parenthèses (il est possible d'utiliser plusieurs conditions en utilisant des opérateurs booléens), suivie du bloc de code que nous souhaitons exécuter si la condition est remplie. Il est également possible d'utiliser le mot clé else (impérativement placé apres un if) afin d’exécuter un bloc de code si la condition n'a pas été remplie. Il existe également un autre type de condition : le switch case, dont l'utilisation est moins rependue.

Exemple :

boolean test = true ;
int nombre = 6 ;
void setup() {
if(test == true) {
	println(« test est vrai ! ») ;
}
else {
	println(« test est faux ! ») ;
}
if(nombre ≥ 6 && test!= false) {
println(« nombre est supérieur ou égal à 6 et test différent de faux ») ;
}

Les boucles sont des éléments essentiels pour coder efficacement : elle permettent d'automatiser les tâches répétitives et donc de rendre plus clair notre code. Il existe plusieurs type de boucle : le while(*condition*) suivi de sa condition entre parenthèse puis du bloc de code a exécuter en boucle tant que la condition est vérifiée. Le for(*initialisation* ;*condition*;incrémentation*) permet d’exécuter un bloc de code tant que la variable locale associée à la boucle vérifiera une condition sachant qu'elle pourra être modifiée à chaque tours de boucle.

Exemple :

int test = 10 ;
void Setup() {
	While(test >= 5) {
		for(int i = 0 ; i > 5 ; i++) {
			println(« Je suis dans le for à la valeur i =  » + i) ;
			test – ;
		}
		println(« Je suis dans le while car test ≥ 5 ») ;
		test – ;
	}
}

Les Fonctions

La fonction permet d'isoler un bloc de code et d’appeler ce celui ci à n'importe quel moment, autant de fois que l'on le souhaite. Elle est définie par 3 éléments :

  • Un type : void si elle ne retourne aucune valeur ou un type de variable pour retourner une variable du même type avec l'instruction return *variable*
  • Un nom : défini par l'utilisateur à condition que celui-ci ne soit pas déjà utilisé, il permettra d’appeler la fonctionAttention, certains nom sont déjà occupé de base sur Processing tel que Size() pour régler la taille de la fenêtre ou rect() pour tracer des rectangles.
  • (Optionnel) Des paramètres : ce sont des variables qui pourront intéragir avec le bloc de code afin de rendre la fonction plus polivalente.

Exemple :

void Setup(){
	affiche(« Bonjour ») ;
}
void affiche(String t) {
	println(t) ;
}

Conclusion

Nous n'avons vu ici, qu'une petite partie de ce qu'il est possible de faire avec processing et il existe de nombreuses ressources en ligne dont particuliairement la référence qui liste et explque (en anglais) toutes les instruction de processing. De plus d'autres languages (le javascript, le java ou le C) utilisent un syntaxe relativement proche de celle du Processing, il peut donc âtre un bon point de départ à la découverte du monde la programmation !
