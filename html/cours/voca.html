Vocabulaire Informatique

Le Fichier

Un fichier stocke des informations sous forme de séquence d’octets (soit une séquence de binaire : “de 0 ou de 1”). Il en existe plusieurs type selon les informations à y stocker :

  • Des programmes : une séquence d’instructions que l’ordinateur pourra comprendre et exécuter.
  • Des images, de la vidéo, du son ou du texte issu d’un traitement de texte : ces fichiers sont spécialement encodés pour être lu par certains logiciels. On peut par exemple ouvrir les images avec Gimp ou ouvrir du son avec VLC.
  • Du texte brut : contient simplement une suite de caractères.

Afin de savoir à quel type de fichier nous ou l’ordinateur est confronté, il existe des indicateurs dans les les métadonnés du fichier et l’extension du fichier. L’ordinateur peut les reconnaître et choisir le logiciel avec lequel exploiter ce fichier. Il existe donc de très nombreuses extensions de fichier, en voici quelques uns :

  • *.exe, *.bat : les fichers executables
  • *.jpeg, *.png : pour les images
  • *.mp3, *.flac : pour l’audio
  • *.mp4, *.flv : pour la vidéo
  • *.doc, *.odt : pour le traitement de texte
  • *.txt, *.log : pour le texte brut
  • ...

Le Logiciel

Le logiciel est l'ensemble de programmes, procédures et ressources nécessaires au fonctionnement d'un système informatique. Ils sont créés par des développeur pour les utilisateurs d’ordinateur ou de tout autre système informatisé. L'exécution d’un programme permet d’effectuer une tâche. Ainsi lorsque nous utilisons notre ordinateur, nous nous servons d’une multitude de logiciel, par exemple Firefox pour naviguer sur internet ou encore l’explorateur de fichiers pour accéder au données stockés sur l’ordinateur.

L'Archive

L’archive est un fichier pouvant contenir plusieurs autres fichiers ou même une arborescence. Elle peut être compressée (c’est à dire qu’elle prend moins de place sur le périphérique sur lequel elle est stockée) et cryptée (nécessitant un mot de passe pour accéder aux fichiers). Afin d’exploiter ce type de fichier, l’utilisateur d’un ordinateur peut utiliser différents logiciels : WinRar, Winzip (Windows), 7zip (multi-plateforme), tar (Linux) et même l’Explorateur de Windows peut extraire des archives. Les archives se présentent sous différents formats ayant tous leurs avantages et inconvénients : *.rar, *.zip, *tar.gz, *.jar (pour les machines virtuelles Java), *.iso (image de disque pour logiciels).

L’archive est donc un moyen un moyen de faciliter le déplacement, le stockage de donnés mais aussi d’en assurer leur protection grâce à un mot de passe.
