HTML5 & CSS3

Le HTML,qu'est ce que c'est ?

Le HTML (HyperText Markup Language) est un format de présentation de données permettant de créer des pages web pouvant être lues dans des navigateurs.

En 1991, Tim Berners-Lee met au point ce language pour mettre en forme un document, le logiciel qui permet de lire ce fichier et de l'afficher (le navigateur ou browser en anglais) ainsi que le protocole qui permet le transfert des informations. Nous utilisons aujourd'hui le HTML5.

C'est un langage de description de données, et non un langage de programmation. Il est figé c'est-à-dire qu'une fois le document chargé dans le navigateur, il ne répond à aucune action de l'utilisateur sur le contenu de la page.

Ce langage est pourvu d'un système de balisage qui va permettre de structurer notre document.

Les Balises HTML

Une balise HTML est un élément que l'on va ajouter au texte de départ pour dire au navigateur de quelle manière l'afficher. Elle n'est pas affichée telle quelle dans le navigateur, elle est interprétée par celui-ci. Elle est toujours délimitée par les signes < et >.

Exemple :

<html>

Une balise peut en plus comporter de zéro à plusieurs attributs. Les attributs sont des informations complémentaires qui la caractérisent. Ils se présentent sous la forme nomattribut="valeur".

Exemple :

<html lang="fr">

Chaque balise ouverte doit être fermée, cependant il existe des exceptions.

    Il existe 2 type de balises :
  • Les balises simples : Ce sont des balises qui sont dites "vides", c'est-à-dire qu'elles ne vont contenir aucune autre balise HTML. Ces balises n'ont pas besoin d'être fermées.
  • Les balises doubles : Les balises doubles sont dites ouvrantes/fermantes, c'est-à-dire qu'elles nécessitent deux balises, une ouvrante et une fermante dans lesquelles on va pouvoir mettre d'autres balises ou du texte. La balise fermante est identique à la balise ouvrante, à la différence qu'elle contient un "/" pour indiquer à quel endroit on la ferme.

Exemple :

<p>Balise double</p>

<img src="balise simple">

Il existe de nombreuses balises ayant chacune une utilitée différente, vous pouvez les retrouver dans des listes comme celle-ci.

La structure de base

La première ligne du document s'appelle le DOCTYPE. C'est une balise indispensable pour conserver la compatibilité du rendu de votre page sur les différents navigateurs modernes.

Après le DOCTYPE, vient la balise <html>. Elle encadre l'ensemble des autres balises.

La balise <html> contient deux grandes balises : <head>, qui va contenir tout l'en-tête de la page et <body> qui va contenir tout le corps du document. Ces balises ne doivent apparaitre qu'une seule et unique fois dans la page

Exemple :

<!DOCTYPE html>
<html>
<head>
    <title>Titre de la page</title>
    <meta charset=utf-8">
</head>
<body>
 
<!-- Ici votre site -->
 
</body>
</html>

Les commentaires

Les commentaires permettent de laisser des informations ou des descriptions du code et sont utilisables à n'importe quel endroit de votre document. Cependant, ce ne sont pas ces commentaires-là qu'il faudra utiliser pour les langages JavaScript et CSS qui possèdent leur propre système de commentaires. Ces commentaires ne seront pas affichés dans le navigateur mais ils resteront présents dans le code source.

Ils sont délimités par <!-- et --> et peuvent être multilignes.

Exemple :

<!-- Voici un premier commentaire sur une ligne. -->
 
<!-- Voici un deuxième commentaire.
     Celui-ci est multilignes. -->

Le CSS, qu'est ce que c'est ?

Vous aurez surement remarqué qu'une page uniquement réalisée en HTML est un peu "simple", le texte s'affiche en noir sur blanc de manière très sobre. Heureusement il existe un moyen d'embellir (ou pas) la mise en page du site en y appliquant des styles.

Le CSS (pour Cascading Style Sheet) est une méthode très efficace pour les appliquer au HTML. Il existe 3 façons de procéder :

  • Dans le corps du code HTML;
  • Dans l'en-tête de la page;
  • Dans une feuille de style séparée du code HTML (méthode conseillée)

Le CSS a été introduit au milieu des années 1990, CSS devient couramment utilisé dans la conception de sites web et bien pris en charge par les navigateurs web dans les années 2000. Nous utilisons aujourd'hui le CSS3

Utilisation d'une feuille de style

Cette méthode compte de nombreux avantages : elle permet à plsuieur pages HTML de n'utiliser qu'une seule feuille de style ou de pouvoir définir plusieurs styles différent sur une seule page HTML e nfonction des besoins, ce qui permet de réduire la taille des fichiers et faciliter les mises à jour.

Pour inclure une feuille de style dans une page HTML on fait appel à la balise <link href="mes-styles.css" rel="stylesheet" type="text/css" /> on indiquera dans le href le chemin pour accéder à la feuille de style.

La syntaxe du CSS3

Quelques règles suffisent pour pouvoir séparer le contenu et sa présentation. L'essentiel est de comprendre comment établir le lien entre un style et la phrase à laquelle s'applique le style. Ce mécanisme est heureusement assez simple.

Avec les CSS, vous pouvez changer la présentation de toutes les balises (X)HTML standards. La syntaxe de présente de cette façon :

  • Préciser le nom de la balise en premier : c'est le sélecteur
  • Encadrer les propriétés par des accolades { } ;
  • Placer un point virgule ; derrière chaque propriété.

Exemple :

Dans le fichier style.css :

p{
	color: blue;
	font-weight: bold;
}

Dans le fichier HTML

	<!DOCTYPE html>
	<html>
	<head>
		<title>Test</title>
		<link rel="stylesheet" type="text/css" href="style.css">
	</head>
	<body>
		<p>Je suis stylisé par la feuille de style</p>
		<p>Moi aussi !</p>
		<span>Mais pas moi</span>
	</body>
	</html>

Dans cet exemple, tous le texte contenu dans les balise <p> auront une couleur bleu et seront en gras.

Il existe bien sûr de nombreuses autres propriétés pour modifier le HTML que vous pouvez retrouver ici, il existe égalment de nombreux autres sélecteur dont une liste est disponible ici.

Conclusion

Cette page n'est qu'un apperçu de ce qu'il est possible de réaliser avec le HTML et le CSS. Ces languages sont en constente évolution avec l'ajout de plus en plus de fonctionnalité (support des terminaux mobiles, drag and drop ...) car c'est une technologie essentielle au fonctionnement du web !

Il est également possible d'améliorer l'expérience du visiteur en utilisant le Javascript qui permet d'animer les pages web. A l'aide de ces 3 trois technologie, les possibilités sont inifnies !
